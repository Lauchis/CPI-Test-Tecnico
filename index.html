Highlights

Grind your part in a single set-up

The LG performs multiple grinding and measuring operations in a single set-up, so you can start and finish your part in the same cycle. Combine several grinding wheelheads and avoid stopping the machine for each opeartion. which ones do you choose?

External, faces, cones, unround, contours, internal...

If you need to grind multiple parts on a single machine, the LG has also elements to change references quickly: movable tailstock, automation systems, counter heads...
